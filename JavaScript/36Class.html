OOPS

Object Oriented Programming structure/system

based on 4 pillars

  1. Encapsulation => wrapping the data into single unit(its done through class, class is an example of Encapsulation)
  2. Inheritance (access parent data in child)
  3. Polimorphisum (many form same things overloading/overrriding)
  4. Abstraction (hide something method will not be contain body/defination at declaration )
